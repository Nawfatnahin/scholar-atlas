# Version 2.0.1 — 2026-05-27

## Summary
Restored full functionality of the `SemesterProgressWidget` overview card on the main dashboard. Corrected invalid queries that attempted to reference non-existent `user_settings` and `subjects` table columns. Updated the queries to pull target CGPA dynamically from `cgpa_settings` and attendance metrics using `attendance_records`. Introduced dynamic semester duration and start date estimation based on earliest subject course schedules, and integrated high-fidelity cumulative CGPA forecasting calculation inside the widget to perfectly synchronize the dashboard overview stats with the CGPA Manager page.
