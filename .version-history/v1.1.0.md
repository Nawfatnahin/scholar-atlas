# Version 1.1.0 — 2026-05-27

## Summary
Introduced high-fidelity **Extra Classes** logging into the Attendance Tracker to allow logging, viewing, and deleting manual class sessions that occurred outside of the regular schedule, with automatic real-time stats updates. Corrected theme initialization default behavior so that shared links or new visitors always open in light mode by default, preventing dark-mode flashes or overrides unless explicitly chosen. Aligned the instruction button popups in the Attendance and Task pages to display only fully available features, removing placeholders for non-existent simulators and sorting mechanics.
