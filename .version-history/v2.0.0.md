# Version 2.0.0 — 2026-05-27

## Summary
Redesigned the CGPA Manager section of Scholar Atlas into a fully semester-aware system. Introduced an automated 4-step initialization wizard (InitManagerModal) to set degree lengths, active semesters, and cached GPAs for previous semesters. Created tabbed semester views (SemesterTabs) allowing users to switch fluidly between academic terms, alongside clean, animated completion bars (DegreeProgressBar) and a summary panel (CGPASummaryCard) displaying real-time CGPA metrics, total credits, distance from target, and customized status notifications. Added high-fidelity dark and light theme compliance across all components, a context-aware instructions button, custom per-semester targets, and database migrations to track metadata reliably.
