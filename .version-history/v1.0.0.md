## Database Tables (Supabase)
- users (via Supabase Auth)
- subjects (id, user_id, name, color, created_at, course_code, semester_start_date, total_weeks)
- attendance_logs (id, subject_id, status, date)
- attendance_records (computed/referenced in UI)
- class_sessions (id, subject_id, user_id, date, status, is_extra)
- tasks (id, user_id, subject_id, title, due_date, status, priority)
- cgpa_settings (id, user_id, target_cgpa)
- cgpa_entries (id, user_id, cgpa, created_at)
- cgpa_courses_manual / cgpa_courses_auto (per subject mark tracking)
- holidays (id, user_id, name, date)
