## Core Upgrades

### 1. ASRP v2.0 — Autonomous Skill Routing Protocol with Permission Gate
- JARVIS now silently monitors every request and matches it against a 15-skill trigger map
- Permission is **always requested** before skill activation — no more auto-fire
- Permission format: *"Detecting [X], Sir. I would like to activate [skill] — [plan]. Shall I proceed, Sir?"*
- Confidence threshold: ≥70% match required before surfacing
- Multi-skill conflict resolver: selects most specific skill, surfaces both if ambiguous
- Session-wide override: saying "permitted" suspends gate for full session
