Everything a student needs.
Nothing they don't.

Three focused tools that solve the three biggest pain points in student life — PDF chaos, forgotten attendance and missed deadlines.

PDF Tools

Stop emailing files to yourself. Merge lecture slides, split chapters, convert to images — all processed in seconds with a clean download link.

  • Merge multiple PDFs into one
  • Split PDFs by page range
  • Convert PDF pages to images
  • Compile images into a PDF
  • Personal file history

Attendance Tracker

Know exactly where you stand before it's too late. Track sessions per subject, get warned when you drop below 60% and plan your absences smartly.

  • Create subjects with colour labels
  • Mark present / absent per session
  • Auto-calculated percentage
  • Warning at <60% threshold
  • Per-subject session calendar

Task Tracker

A lightweight assignment board designed for student workflows. Set priorities, link tasks to subjects and see what's due at a glance.

  • Create tasks with due dates
  • Priority levels (Low / Med / High)
  • Kanban or list view
  • Filter by subject or status
  • One-click task completion
